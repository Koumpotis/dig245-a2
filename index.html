How to view the HTML source of a Website on Chrome

There are 4 steps you need to take in order to "reveal the magician's secrets"

  1. Open the browser (Google Chrome).
  2. Go to the page you wish to see the HTML code for.
  3. Right-click on the page.
  4. Click 'View page source.'
